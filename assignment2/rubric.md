Rubric

Introduction Requirements (10)

Not too important. It should be there.

Assumption Requirements (30)

Assumptions should not directly contradict statements made about the system in the assignment. For example, the assignment states that “All offices are networked via “Cloud Connect””, so they should not assume that another company networks the offices.
Findings and Recommendations Requirements (60)

All findings should have a matching recommendation, even if that recommendation is to make no change.
There should be at least 4 findings and recommendations.
The findings should describe scenarios which are at least remotely possible.
The set of assumptions should be complete with respect to the Findings and Recommendation section. That is, each finding should be based on information contained in the assignment document, or information in the assumptions.
